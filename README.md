# The Carbon Cost of Prosperity: Examining the Development–Emission Tradeoff

This project investigates the relationship between economic development and carbon emissions by constructing a gloabal dataset on GDP per capita and CO₂ emissions per capita. It explores whether higher income levels are associated with higher per capita CO₂ emissions and quantifies this relationship using a log–log regression framework.

The analysis highlights the structural link between economic growth and environmental impact, while also emphasizing global inequality in emissions and the challenges of sustainable development.


## Key Findings  
- Strong positive relationship between GDP per capita and CO₂ emissions per capita.
- A 1% increase in GDP per capita is associated with a ~1.10% increase in emissions.
- Significant inequality in global emissions distribution with Higher-income oil producing countries dominating per capita emissions

## Data Sources  
- CO₂ emissions per capita: Our World in Data (OWID).
- GDP per capita: World Bank

## Methods  
- Data cleaning and merging (Our World in Data, World Bank)  
- Log–log regression analysis  
- Correlation analysis (Pearson, Spearman, Kendall)  
- Residual diagnostics (Shapiro–Wilk test)  
- Data visualization using `ggplot2` and `plotly`
